Etiquetas para crear formularios




ul

La etiqueta (ul) define una lista no ordenada en HTML, donde los elementos de la lista se presentan con viñetas (puntos) por defecto. Se utiliza para agrupar una colección de elementos de manera que no haya un orden específico de prioridad o secuencia. Dentro de (ul), cada elemento de la lista se coloca dentro de una etiqueta (li). Es ideal para listas de ítems que no requieren un orden numérico, como características o tareas.

Sintaxis Ejemplo
<ul > 
                                        <html>
                                        <head>
                                            <title>Ejemplo de UL</title>
                                        </head>
                                        <body>
                                            <ul>
                                                <li>Elemento de lista 1</li>
                                                <li>Elemento de lista 2</li>
                                                <li>Elemento de lista 3</li>
                                            </ul>
                                        </body>
                                        </html>



ol

La etiqueta (ol) se utiliza para crear una lista ordenada, donde los elementos se presentan con números o letras que indican su posición secuencial. Cada elemento de la lista se coloca dentro de una etiqueta (li) y se numera automáticamente por el navegador. Es útil cuando el orden de los elementos es importante, como en instrucciones paso a paso o listas de clasificación. También se puede personalizar el tipo de numeración con el atributo type.

Sintaxis Ejemplo
<ol> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de OL</title>
                                        </head>
                                        <body>
                                            <ol>
                                                <li>Elemento ordenado 1</li>
                                                <li>Elemento ordenado 2</li>
                                                <li>Elemento ordenado 3</li>
                                            </ol>
                                        </body>
                                        </html>



li

La etiqueta (li) define un elemento dentro de una lista, ya sea ordenada (ol) o no ordenada (ul). Cada (li) representa un ítem individual en la lista y puede contener texto, enlaces, imágenes u otros elementos HTML. Es la unidad básica que compone el contenido de las listas en HTML. Las listas anidadas se crean colocando (ul) o (ol) dentro de un (li), creando así sublistas. Es una herramienta clave para estructurar contenido en listas.

Sintaxis Ejemplo
<li > 
                                 <html>
                                 <head>
                                     <title>Ejemplo de LI</title>
                                 </head>
                                 <body>
                                     <ul>
                                         <li>Elemento de lista 1</li>
                                         <li>Elemento de lista 2</li>
                                         <li>Elemento de lista 3</li>
                                     </ul>
                                 </body>
                                 </html>



dl

La etiqueta (dl) se utiliza para definir una lista de definición, que es un tipo especial de lista en HTML. Esta lista consta de pares de términos y sus descripciones. Dentro de (dl), se agrupan términos (dt) y descripciones (dd), lo que es útil para glosarios, diccionarios, o listas de características y valores. No tiene un formato visual predeterminado, pero se utiliza comúnmente para listas donde los elementos necesitan explicación adicional.

Sintaxis Ejemplo
<dl> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de DL</title>
                                        </head>
                                        <body>
                                            <dl>
                                                <dt>Término 1</dt>
                                                <dd>Descripción del término 1</dd>
                                                <dt>Término 2</dt>
                                                <dd>Descripción del término 2</dd>
                                            </dl>
                                        </body>
                                        </html>



dt

La etiqueta (dt) define un término en una lista de definición (dl). Se utiliza para representar el término que será definido o descrito por el siguiente elemento (dd). En un glosario, por ejemplo, (dt) podría contener una palabra que luego se explica en el (dd) correspondiente. Es parte del par clave que forma una lista de definición, y se utiliza para identificar el elemento que se está definiendo.

Sintaxis Ejemplo
<dt> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de DT</title>
                                        </head>
                                        <body>
                                            <dl>
                                                <dt>Término 1</dt>
                                                <dd>Descripción del término 1</dd>
                                                <dt>Término 2</dt>
                                                <dd>Descripción del término 2</dd>
                                            </dl>
                                        </body>
                                        </html>



dd

La etiqueta (dd) se utiliza para proporcionar la descripción o definición de un término definido por (dt) en una lista de definición (dl). Aparece justo después del término al que describe y generalmente se indenta para diferenciarse visualmente. Es útil para proporcionar detalles adicionales o explicaciones sobre el término presentado en (dt). Las listas de definición son ideales para glosarios, términos y condiciones, o especificaciones de productos.

Sintaxis Ejemplo
<dd > 
                                        <html>
                                        <head>
                                            <title>Ejemplo de DD</title>
                                        </head>
                                        <body>
                                            <dl>
                                                <dt>Término 1</dt>
                                                <dd>Descripción del término 1</dd>
                                                <dt>Término 2</dt>
                                                <dd>Descripción del término 2</dd>
                                            </dl>
                                        </body>
                                        </html>



form

La etiqueta form se utiliza para crear un formulario en una página web, permitiendo a los usuarios enviar datos al servidor. Un formulario puede incluir diversos controles de entrada como campos de texto, botones de opción, casillas de verificación y más. Dentro de form, se pueden definir atributos como action (que especifica la URL a la que se envían los datos del formulario) y method (que define el método HTTP a utilizar, como GET o POST).

Sintaxis Ejemplo
<form > 
                                        <html>
                                    <head>
                                        <title>Ejemplo de Formulario</title>
                                    </head>
                                    <body>
                                        <form action="/submit" method="post">
                                            <fieldset>
                                                <legend>Información Personal</legend>
                                                <label for="name">Nombre:</label>
                                                <input type="text" id="name" name="name" required />
                                                <br />
                                                <label for="email">Correo Electrónico:</label>
                                                <input type="email" id="email" name="email" required />
                                                <br />
                                                <label for="message">Mensaje:</label>
                                                <textarea id="message" name="message"></textarea>
                                                <br />
                                                <input type="submit" value="Enviar" />
                                            </fieldset>
                                        </form>
                                    </body>
                                    </html>



input

La etiqueta input se utiliza para crear campos de entrada dentro de un formulario. Puede ser configurada para aceptar diferentes tipos de datos según el valor del atributo type, como texto, contraseñas, correos electrónicos, números y botones.

Sintaxis Ejemplo
< input> 
                                        <html>
                                    <head>
                                        <title>Ejemplo de Campo de Entrada</title>
                                    </head>
                                    <body>
                                        <form action="/submit" method="post">
                                            <label for="username">Nombre de Usuario:</label>
                                            <input type="text" id="username" name="username" placeholder="Ingresa tu nombre de usuario" required />
                                            <br />
                                            <label for="password">Contraseña:</label>
                                            <input type="password" id="password" name="password" placeholder="Ingresa tu contraseña" required />
                                            <br />
                                            <label for="email">Correo Electrónico:</label>
                                            <input type="email" id="email" name="email" placeholder="Ingresa tu correo electrónico" required />
                                            <br />
                                            <label for="age">Edad:</label>
                                            <input type="number" id="age" name="age" min="0" max="120" placeholder="Ingresa tu edad" />
                                            <br />
                                            <label for="subscribe"><input type="checkbox" id="subscribe" name="subscribe" /> Suscribirse a boletín</label>
                                            <br />
                                            <input type="submit" value="Enviar" />
                                        </form>
                                    </body>
                                    </html>



textarea

La etiqueta textarea se utiliza para crear un área de texto en un formulario, permitiendo a los usuarios ingresar texto de varias líneas. A diferencia del campo input, que está limitado a una sola línea, textarea puede expandirse para acomodar más texto. Se pueden definir atributos como rows (para especificar la altura en términos de líneas de texto) y cols (para definir el ancho en términos de caracteres). Esta etiqueta es útil cuando se necesita que el usuario proporcione comentarios largos o descripciones detalladas.

Sintaxis Ejemplo
< textarea> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de Área de Texto</title>
                                        </head>
                                        <body>
                                            <form action="/submit" method="post">
                                                <label for="comments">Comentarios:</label>
                                                <textarea id="comments" name="comments" rows="4" cols="50" placeholder="Escribe tus comentarios aquí"></textarea>
                                                <br />
                                                <input type="submit" value="Enviar" />
                                            </form>
                                        </body>
                                        </html>



label

La etiqueta label se utiliza para asociar un texto descriptivo con un control de entrada en un formulario, como un campo de texto o un botón de opción. El atributo for de label se usa para enlazar la etiqueta con un campo específico mediante el ID del campo, mejorando la accesibilidad y la usabilidad del formulario.

Sintaxis Ejemplo
<label > 
                                        <html>
                                    <head>
                                        <title>Ejemplo de Opciones en Menú Desplegable</title>
                                    </head>
                                    <body>
                                        <form action="/submit" method="post">
                                            <label for="fruit">Elige una fruta:</label>
                                            <select id="fruit" name="fruit">
                                                <option value="apple">Manzana</option>
                                                <option value="banana">Plátano</option>
                                                <option value="cherry">Cereza</option>
                                                <option value="date">Dátil</option>
                                            </select>
                                            <br />
                                            <input type="submit" value="Enviar" />
                                        </form>
                                    </body>
                                    </html>



fieldset

La etiqueta fieldset se utiliza para agrupar elementos dentro de un formulario, agrupando controles relacionados y sus etiquetas en una sección coherente. Esto ayuda a organizar mejor el contenido del formulario y a mejorar la accesibilidad. Dentro de un fieldset, se pueden incluir controles de entrada, etiquetas y otros elementos del formulario.

Sintaxis Ejemplo
<fieldset > 
                                        <html>
                                    <head>
                                        <title>Ejemplo de Etiqueta</title>
                                    </head>
                                    <body>
                                        <form action="/submit" method="post">
                                            <label for="username">Nombre de Usuario:</label>
                                            <input type="text" id="username" name="username" placeholder="Ingresa tu nombre de usuario" required />
                                            <br />
                                            <label for="email">Correo Electrónico:</label>
                                            <input type="email" id="email" name="email" placeholder="Ingresa tu correo electrónico" required />
                                            <br />
                                            <label for="message">Mensaje:</label>
                                            <textarea id="message" name="message" rows="4" cols="50" placeholder="Escribe tu mensaje aquí"></textarea>
                                            <br />
                                            <input type="submit" value="Enviar" />
                                        </form>
                                    </body>
                                    </html>



legend

La etiqueta legend se usa dentro de un fieldset para proporcionar una descripción o título para el grupo de controles agrupados. El texto dentro de legend actúa como una etiqueta para el contenido del fieldset, ayudando a los usuarios a comprender el propósito del grupo de controles.

Sintaxis Ejemplo
<legend > 
                                        <html>
                                        <head>
                                            <title>Ejemplo de Fieldset</title>
                                        </head>
                                        <body>
                                            <form action="/submit" method="post">
                                                <fieldset>
                                                    <legend>Información Personal</legend>
                                                    <label for="name">Nombre:</label>
                                                    <input type="text" id="name" name="name" placeholder="Ingresa tu nombre" required />
                                                    <br />
                                                    <label for="email">Correo Electrónico:</label>
                                                    <input type="email" id="email" name="email" placeholder="Ingresa tu correo electrónico" required />
                                                    <br />
                                                    <label for="age">Edad:</label>
                                                    <input type="number" id="age" name="age" min="0" max="120" placeholder="Ingresa tu edad" />
                                                </fieldset>
                                                <br />
                                                <input type="submit" value="Enviar" />
                                            </form>
                                        </body>
                                        </html>



button

La etiqueta button se utiliza para crear un botón en una página web que los usuarios pueden hacer clic. Los botones pueden tener diversos propósitos, como enviar un formulario, realizar una acción o desencadenar un evento. El atributo type puede tener valores como submit (para enviar un formulario), reset (para restablecer los campos del formulario) o button (para ejecutar una acción específica mediante JavaScript). El texto o contenido dentro de la etiqueta button se muestra como la etiqueta del botón, proporcionando una indicación clara de la acción que realiza.

Sintaxis Ejemplo
<button> 
                                        <html>
                                    <head>
                                        <title>Ejemplo de Legend</title>
                                    </head>
                                    <body>
                                        <form action="/submit" method="post">
                                            <fieldset>
                                                <legend>Información de Contacto</legend>
                                                <label for="name">Nombre:</label>
                                                <input type="text" id="name" name="name" placeholder="Ingresa tu nombre" required />
                                                <br />
                                                <label for="email">Correo Electrónico:</label>
                                                <input type="email" id="email" name="email" placeholder="Ingresa tu correo electrónico" required />
                                                <br />
                                                <label for="phone">Teléfono:</label>
                                                <input type="tel" id="phone" name="phone" placeholder="Ingresa tu número de teléfono" />
                                            </fieldset>
                                            <br />
                                            <input type="submit" value="Enviar" />
                                        </form>
                                    </body>
                                    </html>



datalist

La etiqueta datalist se utiliza para proporcionar una lista de opciones predefinidas para un campo de entrada (input). Cuando se usa con input, permite al usuario seleccionar una opción de una lista desplegable o ingresar un valor personalizado. Las opciones se definen dentro de datalist mediante etiquetas option.

Sintaxis Ejemplo
<datalist> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de Botón</title>
                                        </head>
                                        <body>
                                            <form action="/submit" method="post">
                                                <label for="name">Nombre:</label>
                                                <input type="text" id="name" name="name" placeholder="Ingresa tu nombre" required />
                                                <br />
                                                <label for="email">Correo Electrónico:</label>
                                                <input type="email" id="email" name="email" placeholder="Ingresa tu correo electrónico" required />
                                                <br />
                                                <button type="submit">Enviar</button>
                                                <button type="reset">Restablecer</button>
                                            </form>
                                        </body>
                                        </html>



optgroup

La etiqueta optgroup se usa dentro de un elemento select para agrupar opciones relacionadas en un menú desplegable. Esto ayuda a organizar las opciones en categorías o secciones, facilitando la navegación por el menú desplegable.

Sintaxis Ejemplo
< optgrup> 
                                        <html>
                                    <head>
                                        <title>Ejemplo de Datalist</title>
                                    </head>
                                    <body>
                                        <form action="/submit" method="post">
                                            <label for="fruit">Elige una fruta:</label>
                                            <input list="fruits" id="fruit" name="fruit" placeholder="Selecciona una fruta" />
                                            <datalist id="fruits">
                                                <option value="Manzana">
                                                <option value="Plátano">
                                                <option value="Cereza">
                                                <option value="Dátil">
                                                <option value="Higo">
                                            </datalist>
                                            <br />
                                            <input type="submit" value="Enviar" />
                                        </form>
                                    </body>
                                    </html>



type="radio"

Cada botón de opción dentro del mismo grupo debe compartir el mismo valor de atributo name para que sólo una opción pueda ser seleccionada a la vez.

                                                    </label>

                                                    </label>

                                            </form>
                                        </body>
                                        </html>

type="submit"

La etiqueta input con el atributo type="submit" se utiliza para crear un botón de envío dentro de un formulario. Al hacer clic en este botón, se envían los datos del formulario al servidor especificado en el atributo action del formulario. Este tipo de botón suele tener un texto predeterminado como "Submit" o "Enviar", aunque también se puede personalizar mediante el atributo value para cambiar el texto del botón.

Sintaxis Ejemplo
<type ="submit" > 
                                        <html>
                                        <head>
                                            <title>Ejemplo de Botón de Envío</title>
                                        </head>
                                        <body>
                                            <form action="/submit" method="post">
                                                <label for="name">Nombre:</label>
                                                <input type="text" id="name" name="name" placeholder="Ingresa tu nombre" required />
                                                <br />
                                                <label for="email">Correo Electrónico:</label>
                                                <input type="email" id="email" name="email" placeholder="Ingresa tu correo electrónico" required />
                                                <br />
                                                <input type="submit" value="Enviar" />
                                            </form>
                                        </body>
                                        </html>



type="reset"

La etiqueta input con el atributo type="reset" se utiliza para crear un botón que restablece todos los campos de un formulario a sus valores predeterminados o iniciales. Al hacer clic en este botón, se borran las entradas del usuario y se restauran los valores establecidos en el HTML del formulario. El texto del botón puede ser personalizado con el atributo value. Este tipo de botón es útil cuando se quiere proporcionar a los usuarios una forma rápida de limpiar todos los campos del formulario y empezar de nuevo.

Sintaxis Ejemplo
<type="reset"> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de Botón de Reinicio</title>
                                        </head>
                                        <body>
                                            <form action="/submit" method="post">
                                                <label for="name">Nombre:</label>
                                                <input type="text" id="name" name="name" placeholder="Ingresa tu nombre" required />
                                                <br />
                                                <label for="email">Correo Electrónico:</label>
                                                <input type="email" id="email" name="email" placeholder="Ingresa tu correo electrónico" required />
                                                <br />
                                                <input type="submit" value="Enviar" />
                                                <input type="reset" value="Restablecer" />
                                            </form>
                                        </body>
                                        </html>



type="hidden"

La etiqueta input con el atributo type="hidden" se utiliza para crear un campo de entrada que no es visible para el usuario, pero que puede enviar datos al servidor cuando se envía el formulario. Este tipo de campo es útil para incluir datos adicionales o parámetros que deben ser enviados con el formulario sin que el usuario los modifique.

Sintaxis Ejemplo
<type="hidden" > 
                                        <html>
                                        <head>
                                            <title>Ejemplo de Campo Oculto</title>
                                        </head>
                                        <body>
                                            <form action="/submit" method="post">
                                                <input type="hidden" name="userID" value="12345" />
                                                <label for="name">Nombre:</label>
                                                <input type="text" id="name" name="name" placeholder="Ingresa tu nombre" required />
                                                <br />
                                                <label for="email">Correo Electrónico:</label>
                                                <input type="email" id="email" name="email" placeholder="Ingresa tu correo electrónico" required />
                                                <br />
                                                <input type="submit" value="Enviar" />
                                            </form>
                                        </body>
                                        </html>
