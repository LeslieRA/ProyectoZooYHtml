Etiquetas para crear tablas




ul

La etiqueta (ul) define una lista no ordenada en HTML, donde los elementos de la lista se presentan con viñetas (puntos) por defecto. Se utiliza para agrupar una colección de elementos de manera que no haya un orden específico de prioridad o secuencia. Dentro de (ul), cada elemento de la lista se coloca dentro de una etiqueta (li). Es ideal para listas de ítems que no requieren un orden numérico, como características o tareas.

Sintaxis Ejemplo
<ul > 
                                        <html>
                                        <head>
                                            <title>Ejemplo de UL</title>
                                        </head>
                                        <body>
                                            <ul>
                                                <li>Elemento de lista 1</li>
                                                <li>Elemento de lista 2</li>
                                                <li>Elemento de lista 3</li>
                                            </ul>
                                        </body>
                                        </html>



ol

La etiqueta (ol) se utiliza para crear una lista ordenada, donde los elementos se presentan con números o letras que indican su posición secuencial. Cada elemento de la lista se coloca dentro de una etiqueta (li) y se numera automáticamente por el navegador. Es útil cuando el orden de los elementos es importante, como en instrucciones paso a paso o listas de clasificación. También se puede personalizar el tipo de numeración con el atributo type.

Sintaxis Ejemplo
<ol> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de OL</title>
                                        </head>
                                        <body>
                                            <ol>
                                                <li>Elemento ordenado 1</li>
                                                <li>Elemento ordenado 2</li>
                                                <li>Elemento ordenado 3</li>
                                            </ol>
                                        </body>
                                        </html>



li

La etiqueta (li) define un elemento dentro de una lista, ya sea ordenada (ol) o no ordenada (ul). Cada (li) representa un ítem individual en la lista y puede contener texto, enlaces, imágenes u otros elementos HTML. Es la unidad básica que compone el contenido de las listas en HTML. Las listas anidadas se crean colocando (ul) o (ol) dentro de un (li), creando así sublistas. Es una herramienta clave para estructurar contenido en listas.

Sintaxis Ejemplo
<li > 
                                 <html>
                                 <head>
                                     <title>Ejemplo de LI</title>
                                 </head>
                                 <body>
                                     <ul>
                                         <li>Elemento de lista 1</li>
                                         <li>Elemento de lista 2</li>
                                         <li>Elemento de lista 3</li>
                                     </ul>
                                 </body>
                                 </html>



dl

La etiqueta (dl) se utiliza para definir una lista de definición, que es un tipo especial de lista en HTML. Esta lista consta de pares de términos y sus descripciones. Dentro de (dl), se agrupan términos (dt) y descripciones (dd), lo que es útil para glosarios, diccionarios, o listas de características y valores. No tiene un formato visual predeterminado, pero se utiliza comúnmente para listas donde los elementos necesitan explicación adicional.

Sintaxis Ejemplo
<dl> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de DL</title>
                                        </head>
                                        <body>
                                            <dl>
                                                <dt>Término 1</dt>
                                                <dd>Descripción del término 1</dd>
                                                <dt>Término 2</dt>
                                                <dd>Descripción del término 2</dd>
                                            </dl>
                                        </body>
                                        </html>



dt

La etiqueta (dt) define un término en una lista de definición (dl). Se utiliza para representar el término que será definido o descrito por el siguiente elemento (dd). En un glosario, por ejemplo, (dt) podría contener una palabra que luego se explica en el (dd) correspondiente. Es parte del par clave que forma una lista de definición, y se utiliza para identificar el elemento que se está definiendo.

Sintaxis Ejemplo
<dt> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de DT</title>
                                        </head>
                                        <body>
                                            <dl>
                                                <dt>Término 1</dt>
                                                <dd>Descripción del término 1</dd>
                                                <dt>Término 2</dt>
                                                <dd>Descripción del término 2</dd>
                                            </dl>
                                        </body>
                                        </html>



dd

La etiqueta (dd) se utiliza para proporcionar la descripción o definición de un término definido por (dt) en una lista de definición (dl). Aparece justo después del término al que describe y generalmente se indenta para diferenciarse visualmente. Es útil para proporcionar detalles adicionales o explicaciones sobre el término presentado en (dt). Las listas de definición son ideales para glosarios, términos y condiciones, o especificaciones de productos.

Sintaxis Ejemplo
<dd > 
                                        <html>
                                        <head>
                                            <title>Ejemplo de DD</title>
                                        </head>
                                        <body>
                                            <dl>
                                                <dt>Término 1</dt>
                                                <dd>Descripción del término 1</dd>
                                                <dt>Término 2</dt>
                                                <dd>Descripción del término 2</dd>
                                            </dl>
                                        </body>
                                        </html>

table

Esta etiqueta define el contenedor de una tabla en HTML. Todo el contenido de la tabla, incluidos encabezados, filas y celdas, se debe colocar dentro de esta etiqueta. Proporciona una estructura para organizar datos tabulares. Aunque no tiene un estilo predeterminado propio, los navegadores aplican un formato básico que se puede personalizar con CSS.

Sintaxis Ejemplo
<table > 
                                <html>
                                <head>
                                    <title>Ejemplo de TABLE</title>
                                </head>
                                <body>
                                    <table border="1">
                                        <tr>
                                            <th>Encabezado 1</th>
                                            <th>Encabezado 2</th>
                                        </tr>
                                        <tr>
                                            <td>Dato 1</td>
                                            <td>Dato 2</td>
                                        </tr>
                                    </table>
                                </body>
                                </html>

Capation

La etiqueta (caption) se utiliza para proporcionar un título o descripción para la tabla. Generalmente, se coloca justo después de la etiqueta (table), y se muestra encima de la tabla. Es útil para describir el contenido o propósito de la tabla. Mejora la accesibilidad al permitir que los usuarios comprendan rápidamente el tema de la tabla. Aunque es opcional, es recomendable en tablas complejas.

Sintaxis Ejemplo
<Caption > 
                                <html>
                                <head>
                                    <title>Ejemplo de CAPTION</title>
                                </head>
                                <body>
                                    <table border="1">
                                        <caption>Título de la tabla</caption>
                                        <tr>
                                            <th>Encabezado 1</th>
                                            <th>Encabezado 2</th>
                                        </tr>
                                        <tr>
                                            <td>Dato 1</td>
                                            <td>Dato 2</td>
                                        </tr>
                                    </table>
                                </body>
                                </html>

Th

Define una celda de encabezado en una tabla, y el texto dentro de (th) suele aparecer en negrita y centrado. Las celdas de encabezado son importantes para describir el contenido de las columnas o filas. Pueden utilizarse en combinación con el atributo scope para mejorar la accesibilidad, indicando si el encabezado se aplica a una fila, columna o grupo. Facilitan la comprensión de la estructura de los datos.

Sintaxis Ejemplo
< > 
                                <html>
                                <head>
                                    <title>Ejemplo de CAPTION</title>
                                </head>
                                <body>
                                    <table border="1">
                                        <caption>Título de la tabla</caption>
                                        <tr>
                                            <th>Encabezado 1</th>
                                            <th>Encabezado 2</th>
                                        </tr>
                                        <tr>
                                            <td>Dato 1</td>
                                            <td>Dato 2</td>
                                        </tr>
                                    </table>
                                </body>
                                </html>

Tr

Representa una fila dentro de una tabla y agrupa celdas de encabezado (th) y celdas de datos (td). Cada fila (tr) organiza los datos en la tabla de manera estructurada. Puede contener una combinación de diferentes tipos de celdas según lo requiera la tabla. Es fundamental para crear la estructura vertical de la tabla. Las filas también pueden ser estilizadas para mejorar la legibilidad.

Sintaxis Ejemplo
< tr > 
                                <html>
                                        <head>
                                            <title>Ejemplo de TH</title>
                                        </head>
                                        <body>
                                            <table border="1">
                                                <tr>
                                                    <th>Encabezado 1</th>
                                                    <th>Encabezado 2</th>
                                                </tr>
                                                <tr>
                                                    <td>Dato 1</td>
                                                    <td>Dato 2</td>
                                                </tr>
                                            </table>
                                        </body>
                                        </html>

Td

Define una celda de datos en una tabla, que es donde se coloca la información principal de la tabla. Estas celdas contienen cualquier tipo de contenido, como texto, imágenes o enlaces. Se colocan dentro de las filas (tr) para organizar los datos de manera lógica. Pueden tener atributos como colspan y rowspan para abarcar múltiples columnas o filas. Son el componente básico para mostrar datos en una tabla.

Sintaxis Ejemplo
< td > 
                                <html>
                                <head>
                                    <title>Ejemplo de TR</title>
                                </head>
                                <body>
                                    <table border="1">
                                        <tr>
                                            <th>Encabezado 1</th>
                                            <th>Encabezado 2</th>
                                        </tr>
                                        <tr>
                                            <td>Dato 1</td>
                                            <td>Dato 2</td>
                                        </tr>
                                    </table>
                                </body>
                                </html>

Thead

Agrupa una o más filas (tr) que contienen celdas de encabezado (th). Se utiliza para separar el encabezado de la tabla del cuerpo (tbody). Facilita la aplicación de estilos y scripts solo a los encabezados. Además, ayuda a mantener el encabezado visible en tablas grandes mientras se desplaza el contenido. Mejora la organización y claridad de la tabla.

Sintaxis Ejemplo
< Thead > 
                                <html>
                                <head>
                                    <title>Ejemplo de THEAD</title>
                                </head>
                                <body>
                                    <table border="1">
                                        <thead>
                                            <tr>
                                                <th>Encabezado 1</th>
                                                <th>Encabezado 2</th>
                                            </tr>
                                        </thead>
                                        <tbody>
                                            <tr>
                                                <td>Dato 1</td>
                                                <td>Dato 2</td>
                                            </tr>
                                        </tbody>
                                    </table>
                                </body>
                                </html>

Tbody

Contiene el cuerpo principal de la tabla, agrupando filas (tr) que tienen celdas de datos (td). Es esencial para separar visualmente y lógicamente los datos del encabezado y pie de tabla. Facilita la aplicación de estilos y scripts específicos al cuerpo de la tabla. Aunque opcional, su uso es recomendado para mejorar la estructura y accesibilidad. Ayuda a organizar grandes cantidades de datos de manera clara.

Sintaxis Ejemplo
< > 
                                <html>
                                <head>
                                    <title>Ejemplo de TBODY</title>
                                </head>
                                <body>
                                    <table border="1">
                                        <thead>
                                            <tr>
                                                <th>Encabezado 1</th>
                                                <th>Encabezado 2</th>
                                            </tr>
                                        </thead>
                                        <tbody>
                                            <tr>
                                                <td>Dato 1</td>
                                                <td>Dato 2</td>
                                            </tr>
                                            <tr>
                                                <td>Dato 3</td>
                                                <td>Dato 4</td>
                                            </tr>
                                        </tbody>
                                    </table>
                                </body>
                                </html>

Tfoot

Agrupa filas (tr) que suelen contener información de resumen, como totales o notas al pie. Se coloca generalmente después del cuerpo de la tabla (tbody), pero puede aparecer antes en el código HTML. Facilita la presentación de datos agregados o finales. Es útil para mejorar la estructura y accesibilidad, permitiendo que el pie de tabla sea visible incluso en tablas largas. Mejora la legibilidad y organización de los datos.

Sintaxis Ejemplo
< Tfoot > 
                                <html>
                                <head>
                                    <title>Ejemplo de TFOOT</title>
                                </head>
                                <body>
                                    <table border="1">
                                        <thead>
                                            <tr>
                                                <th>Encabezado 1</th>
                                                <th>Encabezado 2</th>
                                            </tr>
                                        </thead>
                                        <tbody>
                                            <tr>
                                                <td>Dato 1</td>
                                                <td>Dato 2</td>
                                            </tr>
                                        </tbody>
                                        <tfoot>
                                            <tr>
                                                <td colspan="2">Pie de tabla</td>
                                            </tr>
                                        </tfoot>
                                    </table>
                                </body>
                                </html>

col

Define el estilo para una o varias columnas de la tabla, cuando se coloca dentro de (colgroup). Permite aplicar propiedades de estilo de forma consistente a toda una columna. Facilita la personalización de ancho, color de fondo, o bordes de columnas enteras. Se utiliza para simplificar el manejo de tablas complejas. Es especialmente útil para mantener la consistencia en la presentación de datos.

Sintaxis Ejemplo
< Col > 
                                <html>
                                        <head>
                                            <title>Ejemplo de COL</title>
                                        </head>
                                        <body>
                                            <table border="1">
                                                <colgroup>
                                                    <col style="background-color:lightblue">
                                                    <col style="background-color:lightgreen">
                                                </colgroup>
                                                <tr>
                                                    <th>Encabezado 1</th>
                                                    <th>Encabezado 2</th>
                                                </tr>
                                                <tr>
                                                    <td>Dato 1</td>
                                                    <td>Dato 2</td>
                                                </tr>
                                            </table>
                                        </body>
                                        </html>

Colgroup

Agrupa varios elementos (col), permitiendo la aplicación de estilos a múltiples columnas de la tabla. Facilita la personalización de ancho, color o estilo de varias columnas simultáneamente. Es útil para definir la estructura de columnas antes de que se carguen los datos. Mejora la organización del código HTML al aplicar estilos de manera centralizada. Es esencial para tablas con muchas columnas que comparten propiedades comunes.

Sintaxis Ejemplo
< Colgroup > 
                                <html>
                                <head>
                                    <title>Ejemplo de COLGROUP</title>
                                </head>
                                <body>
                                    <table border="1">
                                        <colgroup>
                                            <col style="background-color:lightblue">
                                            <col style="background-color:lightgreen">
                                        </colgroup>
                                        <tr>
                                            <th>Encabezado 1</th>
                                            <th>Encabezado 2</th>
                                        </tr>
                                        <tr>
                                            <td>Dato 1</td>
                                            <td>Dato 2</td>
                                        </tr>
                                    </table>
                                </body>
                                </html>
