Etiquetas para la base del documento





!DOCTYPE

Se utiliza para informar al navegador web sobre la versión del lenguaje HTML que se está utilizando en un documento. Es una instrucción que debe estar en la primera línea del documento HTML antes de cualquier otra etiqueta. Su función principal es asegurar que el navegador renderice la página de acuerdo con los estándares del HTML especificado.

Sintaxis Ejemplo
<!DOCTYPE html> 
                                    <!DOCTYPE html>
                                    <html>
                                    <head>
                                        <title>Ejemplo de DOCTYPE</title>
                                    </head>
                                    <body>
                                        <h1>¡Hola, mundo!</h1>
                                        <p>Este es un documento HTML5.</p>
                                    </body>
                                    </html>



Html

Se utiliza para informar al navegador web sobre la versión del lenguaje HTML que se está utilizando en un documento. Es una instrucción que debe estar en la primera línea del documento HTML antes de cualquier otra etiqueta. Su función principal es asegurar que el navegador renderice la página de acuerdo con los estándares del HTML especificado.

Sintaxis Ejemplo
<Html> 
                                    <html>
                                     <head>
                                  <title>Ejemplo de HTML</title>
                                    </head>
                                     <body>
                                  <p>Contenido dentro de la etiqueta HTML.</p>
                              </body>
                             </html>



Head

Es un contenedor para los elementos de metadatos y configuraciones del documento HTML. Dentro de (head), se pueden incluir etiquetas que definen el título de la página, enlaces a hojas de estilo externas, scripts, metadatos como el conjunto de caracteres, y otros elementos que no son visibles directamente en la página, pero que son esenciales para la estructura y el comportamiento del documento.

Sintaxis Ejemplo
<Html> 
                                        <html>
                                        <head>
                                        <title>Ejemplo de Head</title>
                                        </head>
                                        <body>
                                        <p>Contenido del cuerpo.</p>
                                        </body>
                                        </html>



Body

Define el cuerpo del documento HTML y contiene todo el contenido que se mostrará en la página web, como texto, imágenes, videos, enlaces, formularios, y otros elementos multimedia e interactivos. Todo lo que está dentro de la etiqueta (body) es lo que el usuario verá en el navegador.

Sintaxis Ejemplo
<Html> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de Body</title>
                                        </head>
                                        <body>
                                            <h1>Título dentro del cuerpo</h1>
                                            <p>Este es el contenido principal.</p>
                                        </body>
                                        </html>



Title

Define el título del documento HTML y se utiliza para especificar el texto que aparecerá en la pestaña del navegador o en el encabezado de la ventana. El título es importante para la identificación de la página y también es utilizado por los motores de búsqueda para indexar y clasificar el contenido de la web.

Sintaxis Ejemplo
<Html> 
                                        <html>
                                        <head>
                                            <title>Este es el título de la página</title>
                                        </head>
                                        <body>
                                            <p>Contenido dentro del cuerpo.</p>
                                        </body>
                                        </html>


Meta

Se utiliza para proporcionar metadatos sobre el documento HTML. Estos metadatos pueden incluir información sobre el conjunto de caracteres, la descripción de la página, palabras clave, autor, y otras configuraciones que ayudan a los navegadores y motores de búsqueda a entender el contenido y las características del documento. Las etiquetas (meta) no se muestran directamente en la página, pero son fundamentales para la optimización y la funcionalidad de la web.

Sintaxis Ejemplo
<Html> 
                                        <html>
                                        <head>
                                            <meta charset="UTF-8">
                                            <meta name="description" content="Ejemplo de meta tag">
                                            <title>Ejemplo de Meta</title>
                                        </head>
                                        <body>
                                            <p>Contenido dentro del cuerpo.</p>
                                        </body>
                                        </html>


Base

Se utiliza para establecer una URL base para todos los enlaces relativos en un documento HTML. Esto significa que si se especifica una URL base, todos los enlaces (enlaces , imágenes , etc.) que utilicen rutas relativas se resolverán en relación con esta URL base. Esto es útil para simplificar la gestión de enlaces en documentos donde se pueden mover los archivos a diferentes directorios.

Sintaxis Ejemplo
<Html> 
                                        <html>
                                        <head>
                                            <base href="https://www.ejemplo.com/">
                                            <title>Ejemplo de Base</title>
                                        </head>
                                        <body>
                                            <p>Enlaces relativos usarán la base especificada.</p>
                                        </body>
                                        </html>


Link

Se utiliza para establecer relaciones entre el documento HTML y otros recursos externos, como hojas de estilo (CSS), iconos, y más. Comúnmente, se utiliza para enlazar hojas de estilo CSS, lo que permite dar formato y diseño a la página web. Esta etiqueta debe colocarse dentro de la sección (head) del documento.

Sintaxis Ejemplo
<Html> 
                                        <html>
                                        <head>
                                            <link rel="stylesheet" href="estilos.css">
                                            <title>Ejemplo de Link</title>
                                        </head>
                                        <body>
                                            <p>Este es un párrafo estilizado con CSS.</p>
                                        </body>
                                        </html>


Style

Se utiliza para definir estilos CSS directamente dentro de un documento HTML. Permite a los desarrolladores especificar reglas de estilo que afectarán a los elementos del documento. Esta etiqueta se coloca dentro de la sección (head) y es útil para estilos que son específicos de una única página, en lugar de enlazar a una hoja de estilo externa.

Sintaxis Ejemplo
<Html> 
                                        <html>
                                        <head>
                                            <style>
                                                body { font-family: Arial, sans-serif; }
                                                h1 { color: blue; }
                                            </style>
                                            <title>Ejemplo de Style</title>
                                        </head>
                                        <body>
                                            <h1>Texto en color azul</h1>
                                        </body>
                                        </html>


Details

Se utiliza para crear un widget que permite a los usuarios mostrar u ocultar información adicional. Se puede utilizar junto con la etiqueta (summary>), que proporciona un encabezado que el usuario puede hacer clic para expandir o contraer el contenido adicional. Esta funcionalidad es útil para mejorar la experiencia del usuario al permitir que se oculte información que no es necesaria en la vista inicial.

Sintaxis Ejemplo
<Html> 
                                        <html>
                                        <head>
                                        <title>Ejemplo de Details</title>
                                        </head>
                                        <body>
                                        <details>
                                        <summary>Más información</summary>
                                        <p>Aquí tienes más detalles sobre el tema.</p>
                                        </details>
                                        </body>
                                        </html>


Summary

Se utiliza dentro de la etiqueta (details) para proporcionar un encabezado que los usuarios pueden hacer clic para expandir o contraer el contenido adicional. El texto dentro de (summary) es visible de forma predeterminada y actúa como un título que describe el contenido que se puede mostrar u ocultar.

Sintaxis Ejemplo
<Html> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de Summary</title>
                                        </head>
                                        <body>
                                            <details>
                                                <summary>Clic para más información</summary>
                                                <p>Este texto aparece cuando se expande.</p>
                                            </details>
                                        </body>
                                        </html>


!--

Se utiliza para crear comentarios en un documento HTML. Los comentarios son notas que los desarrolladores pueden incluir en el código para proporcionar explicaciones, anotaciones o recordatorios. Los comentarios no se muestran en el navegador y son ignorados por el motor de renderizado, lo que los hace útiles para documentar el código sin afectar la presentación o la funcionalidad de la página.

Sintaxis Ejemplo
<Html> 
                                        <html>
                                        <head>
                                            <title>Ejemplo de Comentarios</title>
                                        </head>
                                        <body>
                                            <!-- Este es un comentario en HTML -->
                                            <p>Este es un párrafo visible.</p>
                                        </body>
                                        </html>
