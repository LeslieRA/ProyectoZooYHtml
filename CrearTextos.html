Etiquetas para crear textos




P

La etiqueta (p) se utiliza para definir un párrafo de texto en un documento HTML. Es la forma más común de agrupar y presentar bloques de texto, y se utiliza para estructurar el contenido en fragmentos lógicos y legibles. El texto dentro de un (p) generalmente se muestra con un espacio (margen) antes y después del párrafo para separarlo de otros elementos en la página.

Sintaxis Ejemplo
<P> 
                                            <html>
                                            <head>
                                                <title>Ejemplo de Párrafo</title>
                                            </head>
                                            <body>
                                                <p>Este es un párrafo de texto.</p>
                                            </body>
                                            </html>



H1-H6

Las etiquetas (h1) a (h6) se utilizan para definir encabezados (títulos) de diferentes niveles de importancia en un documento HTML. Los encabezados ayudan a estructurar el contenido jerárquicamente, desde el nivel más alto (h1) hasta el más bajo (h6). Esto mejora tanto la legibilidad para los usuarios como la accesibilidad para los motores de búsqueda y lectores de pantalla.

Sintaxis Ejemplo
<H1-H6> 
                                                <html>
                                                <



Hgroup

En HTML, el elemento hgroup es un contenedor que agrupa encabezados (h1, h2, etc.) con el propósito de jerarquizar y semánticamente agrupar diferentes niveles de encabezados que pertenecen al mismo bloque de contenido. Se utilizaba para situaciones en las que había un encabezado principal seguido de subtítulos relacionados.

Sintaxis Ejemplo
<Hgroup> 
                                               <html>
                                                <head>
                                                  </head>
                                                  <body>
                                                <hgroup>
                                            <h1>Título Principal</h1>
                                         <h2>Subtítulo</h2>
                                       </hgroup>
                                          </body>
                                               </html>



A

La etiqueta (a) se utiliza para definir un hipervínculo en un documento HTML. Un hipervínculo es un enlace que puede dirigir al usuario a otra página web, a una sección diferente dentro de la misma página, a un archivo, o incluso iniciar una acción como enviar un correo electrónico. La etiqueta (a) es esencial para la navegación en la web.

Sintaxis Ejemplo
<a> 
                                                <html>
                                                <head>
                                                    <title>Ejemplo de Enlace</title>
                                                </head>
                                                <body>
                                                    <a href="https://www.ejemplo.com">Visita Ejemplo.com</a>
                                                </body>
                                                </html>



Strong

La etiqueta strong se utiliza para resaltar un fragmento de texto que tiene importancia o énfasis fuerte dentro de un documento HTML. Por defecto, el texto dentro de (strong) se muestra en negrita en la mayoría de los navegadores, lo que visualmente indica su relevancia. Sin embargo, la etiqueta tiene un significado semántico que va más allá de la simple presentación visual, sugiriendo que el texto es de alta importancia.

Sintaxis Ejemplo
<strong> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Strong</title>
                                    </head>
                                    <body>
                                        <p>Este texto es <strong>importante</strong> y debe ser enfatizado.</p>
                                    </body>
                                    </html>



Em

La etiqueta (em) se utiliza para marcar texto que debe ser enfatizado en un documento HTML. Generalmente, el texto dentro de (em) se muestra en cursiva en la mayoría de los navegadores, pero el propósito principal de la etiqueta es añadir un énfasis sutil al contenido, indicando que este texto tiene un significado o tono especial. Semánticamente, (em) sugiere un énfasis moderado, lo cual puede ser interpretado por lectores de pantalla y motores de búsqueda.

Sintaxis Ejemplo
<em> 
                                        <html>
                                        <head>
                                        <title>Ejemplo de Em</title>
                                        </head>
                                        <body>
                                        <p>Este texto está <em>enfatizado</em> para resaltar su importancia.</p>
                                        </body>
                                        </html>



U

La etiqueta (u) se utiliza para subrayar texto en un documento HTML. Aunque antiguamente se usaba exclusivamente para subrayar contenido, en HTML5 se ha redefinido con un propósito más específico: indicar texto que debe diferenciarse visualmente por ser, por ejemplo, un error tipográfico, un nombre propio, o una palabra extranjera. Sin embargo, visualmente, el contenido dentro de (u) sigue apareciendo subrayado en la mayoría de los navegadores.

Sintaxis Ejemplo
<u> 
                                                <html>
                                                <head>
                                                    <title>Ejemplo de U</title>
                                                </head>
                                                <body>
                                                    <p>Este texto está <u>bajo subrayado</u> para indicar énfasis.</p>
                                                </body>
                                                </html>



Del

La etiqueta (del) se utiliza para indicar que un texto ha sido eliminado o que ya no es relevante en el documento HTML. El texto dentro de (del) generalmente se muestra con una línea tachada, lo que visualmente comunica que esa parte del texto ha sido modificada, reemplazada o ya no es válida. Esta etiqueta es útil para mostrar cambios en un documento, como correcciones, actualizaciones, o contenido que se ha descartado.

Sintaxis Ejemplo
<del> 
                                   <html>
                                   <head>
                                       <title>Ejemplo de Del</title>
                                   </head>
                                   <body>
                                       <p>Este texto ha sido <del>eliminado</del> y no es válido.</p>
                                   </body>
                                   </html>



Sup

La etiqueta (sup )se utiliza para definir texto que debe mostrarse como superíndice en un documento HTML. Un superíndice es un texto más pequeño que se coloca ligeramente por encima de la línea base del texto normal. Es comúnmente usado en notaciones matemáticas, científicas, y en referencias como notas al pie.

Sintaxis Ejemplo
<sup> 
                                            <html>
                                            <head>
                                                <title>Ejemplo de Sub</title>
                                            </head>
                                            <body>
                                                <p>El agua es H<sub>2</sub>O.</p>
                                            </body>
                                            </html>



Sub

La etiqueta (sub) se utiliza para definir texto que debe mostrarse como subíndice en un documento HTML. Un subíndice es un texto más pequeño que se coloca ligeramente por debajo de la línea base del texto normal, comúnmente utilizado en fórmulas matemáticas, químicas, y notaciones científicas.

Sintaxis Ejemplo
<sub> 
                                   <html>
                                   <head>
                                       <title>Ejemplo de Sup</title>
                                   </head>
                                   <body>
                                       <p>El área de un cuadrado es lado<sup>2</sup>.</p>
                                   </body>
                                   </html>



Blockquete

La etiqueta (blockquote) se utiliza para definir una cita de un texto extenso. Esta etiqueta indica que el contenido es una cita de otra fuente, y generalmente se presenta con un sangrado a la izquierda (o a la derecha, dependiendo del estilo) para diferenciarlo del texto normal. Es útil para dar contexto a las citas y resaltar su importancia.

Sintaxis Ejemplo
<blockquote> 
                                            <html>
                                            <head>
                                                <title>Ejemplo de Blockquote</title>
                                            </head>
                                            <body>
                                                <blockquote>
                                                    <p>Este es un bloque de cita.</p>
                                                </blockquote>
                                            </body>
                                            </html>



Q

La etiqueta (q) se utiliza para marcar una cita corta dentro de un párrafo. A diferencia de (blockquote), que se usa para citas largas y generalmente se presenta en un bloque separado, (q) se utiliza para citas más breves que se integran directamente en el flujo del texto. Por defecto, el texto dentro de (q) se muestra entre comillas, lo que indica que es una cita.

Sintaxis Ejemplo
<q> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Q</title>
                                    </head>
                                    <body>
                                        <p>Ella dijo: <q>Esto es increíble.</q></p>
                                    </body>
                                    </html>



Pre

La etiqueta (pre) se utiliza para definir texto que debe ser presentado en un formato preformateado. El contenido dentro de (pre) se muestra con un espaciado y una alineación exactos, respetando los espacios en blanco, los saltos de línea y los caracteres de tabulación. Esto es útil para mostrar código fuente, poesía, o cualquier texto donde la presentación precisa sea importante.

Sintaxis Ejemplo
<pre> 
                                                <html>
                                                <head>
                                                    <title>Ejemplo de Pre</title>
                                                </head>
                                                <body>
                                                    <pre>Este es un texto
                                                con saltos de línea
                                                y espacios preservados.</pre>
                                                </body>
                                                </html>



Abbr

La etiqueta (abbr) se utiliza para representar una abreviatura o acrónimo en un documento HTML. Al usar esta etiqueta, se puede proporcionar información adicional sobre el significado de la abreviatura a través del atributo title, que aparece como un tooltip (información emergente) cuando el usuario pasa el cursor sobre la abreviatura. Esto mejora la accesibilidad y comprensión del contenido.

Sintaxis Ejemplo
<abbr> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Abbr</title>
                                    </head>
                                    <body>
                                        <p>La <abbr title="HyperText Markup Language">HTML</abbr> es un lenguaje de marcado.</p>
                                    </body>
                                    </html>



Adress

La etiqueta (address)se utiliza para proporcionar información de contacto sobre una persona o una organización en un documento HTML. Este elemento se suele utilizar para incluir direcciones físicas, direcciones de correo electrónico, números de teléfono, y otros datos relevantes relacionados con el contacto. Por convención, el contenido de address puede estar formateado de manera especial en algunos navegadores, lo que resalta su naturaleza informativa.

Sintaxis Ejemplo
<adress> 
                                                <html>
                                                <head>
                                                    <title>Ejemplo de Address</title>
                                                </head>
                                                <body>
                                                    <address>
                                                        John Doe<br>
                                                        123 Main St.<br>
                                                        Anytown, USA<br>
                                                    </address>
                                                </body>
                                                </html>



Cite

La etiqueta Cite> se utiliza para marcar el título de una obra, como un libro, una película, un artículo, una canción, o cualquier otra creación artística. El contenido dentro de generalmente se presenta en cursiva, indicando que se refiere a un trabajo o una fuente específica. Esta etiqueta ayuda a dar reconocimiento a las obras citadas y proporciona contexto en documentos académicos y literarios.

Sintaxis Ejemplo
<cite> 
                                   <html>
                                   <head>
                                       <title>Ejemplo de Cite</title>
                                   </head>
                                   <body>
                                       <p>Esto es mencionado en <cite>Un libro famoso</cite>.</p>
                                   </body>
                                   </html>
