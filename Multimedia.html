Etiquetas para crear multimedia y scripts





Iframe

La etiqueta (iframe) se utiliza para incrustar otra página web dentro de la página actual. Funciona como un contenedor que muestra un documento HTML independiente, el cual puede provenir del mismo sitio o de un dominio externo. A través de atributos como src, se define la URL del contenido a incrustar, y con width y height, se ajustan las dimensiones del marco. Es útil para integrar contenido externo como videos de YouTube, mapas interactivos, o aplicaciones web. Los iframes pueden afectar el rendimiento y la seguridad, por lo que deben usarse con cuidado.

Sintaxis Ejemplo
<iframe> 
                                      
                        <html>
                        <head>
                      <title>Ejemplo de Iframe</title>
                      </head>
                      <body>
                      <iframe src="https://www.example.com" width="600" height="400" frameborder="0">
                      Este contenido no puede ser mostrado porque tu navegador no soporta iframes.
                      </iframe>
                       </body>
                       </html>



Embed

La etiqueta (embed) se utiliza para incrustar contenido multimedia o aplicaciones en la página web. Es similar a (iframe) pero está más enfocada en incluir objetos que no son HTML, como archivos Flash, videos, o incluso contenido de plugins. El atributo src especifica la URL del recurso a incrustar, mientras que width y height definen su tamaño. Es una etiqueta auto-cerrada y no tiene contenido interno. Es una forma rápida de integrar contenido externo, pero como (iframe), ebe usarse con precaución para evitar problemas de seguridad.

Sintaxis Ejemplo
<Embed> 
                           
                    <html>
                    <head>
                    <title>Ejemplo de Embed</title>
                    </head>
                    <body>
                    <embed src="example-file.pdf" type="application/pdf" width="600" height="400" />
                    <embed src="example-video.mp4" type="video/mp4" width="600" height="400" />
                    </body>
                    </html>



Object

La etiqueta (object) es un contenedor versátil para incrustar cualquier tipo de contenido multimedia, como imágenes, videos, aplicaciones, o documentos PDF. Permite especificar un recurso externo mediante el atributo data, que indica la URL del archivo a mostrar, y type, que define el tipo MIME del recurso. Dentro de (object), se pueden incluir varias etiquetas (param) para proporcionar parámetros adicionales al objeto incrustado. Es más flexible que (embed) y (iframe), permitiendo una mayor integración de contenido interactivo y multimedia.

Sintaxis Ejemplo
<Object> 
                 <html>
                 <head>
                 <title>Ejemplo de Object</title>
                 </head>
                 <body>
                 <object data="example-file.pdf" type="application/pdf" width="600" height="400">
                
Tu navegador no soporta la visualización de PDFs. Puedes descargar el archivo aquí.

                </object>                                 
                <object data="example-video.mp4" type="video/mp4" width="600" height="400">
                
Tu navegador no soporta la visualización de este video. Puedes descargar el archivo aquí.

                 </object>
                </body>
                </html>



Param

La etiqueta (param) se utiliza dentro de (object) para pasar parámetros específicos al recurso incrustado. Cada (param) define un par clave-valor, donde name especifica el nombre del parámetro y value define su valor. Es esencial para controlar cómo se presenta o se comporta el contenido incrustado, permitiendo ajustes como la configuración de un video, la definición de un ID de documento, o la personalización de una aplicación Flash. Aunque es una etiqueta auto-cerrada, es crucial para proporcionar opciones adicionales al objeto.

Sintaxis Ejemplo
<Param> 
     <html>
     <head>
     <title>Ejemplo de Param</title>
     </head>
     <body>
    <object data="example-file.swf" type="application/x-shockwave-flash" width="600" height="400">
     <param name="movie" value="example-file.swf">
    <param name="quality" value="high">
    <param name="bgcolor" value="#ffffff">
    
Tu navegador no soporta el objeto Flash. Puedes descargar el archivo aquí.

     </object>
     </body>
    </html>



Map

La etiqueta (map) se utiliza para definir un mapa de imagen, que es una imagen con áreas seleccionables que actúan como enlaces. Dentro de (map), se definen varias etiquetas (area) que especifican las regiones interactivas de la imagen, asociadas a enlaces mediante el atributo href. Cada (area) se puede definir en forma de rectángulos, círculos, o polígonos para delinear partes específicas de la imagen. El atributo name de (map) se enlaza con el atributo usemap de una imagen para asociar la imagen con el mapa interactivo. Es útil para crear gráficos navegables o imágenes con enlaces específicos.

Sintaxis Ejemplo
<Map> 
    <html>
     <head>
    <title>Ejemplo de Map</title>
    </head>
    <body>
    <img src="example-image.jpg" usemap="#image-map" width="600" height="400" alt="Imagen con mapa de imagen">                        
    <map name="image-map">
    <area shape="rect" coords="34,44,270,350" href="https://www.example.com" alt="Enlace a Example.com">
    <area shape="circle" coords="200,200,50" href="https://www.example.org" alt="Enlace a Example.org">
    <area shape="poly" coords="300,100,400,150,350,200" href="https://www.example.net" alt="Enlace a Example.net">
    </map>
    </body>
    </html>



Area

La etiqueta area se utiliza dentro de un elemento map para definir áreas activas en una imagen de mapa de imagen. Estas áreas son zonas clicables que pueden redirigir a diferentes URLs o activar otras acciones. Cada area define una región de la imagen y su comportamiento asociado, como enlaces o scripts.

Sintaxis Ejemplo
<area> 
    <html>
    <head>
    <title>Ejemplo de Area</title>
    </head>
    <body>
    <img src="example-image.jpg" usemap="#image-map" width="600" height="400" alt="Imagen con mapa de imagen">                                  
    <map name="image-map">
    <area shape="rect" coords="34,44,270,350" href="https://www.example.com" alt="Enlace a Example.com">
    <area shape="circle" coords="200,200,50" href="https://www.example.org" alt="Enlace a Example.org">
    <area shape="poly" coords="300,100,400,150,350,200" href="https://www.example.net" alt="Enlace a Example.net">
    </map>
    </body>
    </html>



Progress

La etiqueta progress se utiliza para mostrar el progreso de una tarea o proceso en curso. Es un elemento visual que presenta una barra de progreso que indica cuánto se ha completado de una tarea. Se usa comúnmente para representar la carga de archivos, el progreso de una descarga o el avance en una operación de procesamiento.

Sintaxis Ejemplo
<Progress> 
                <html>
                 <head>
                 <title>Ejemplo de Progress</title>
                 </head>
                 <body>
                 <h2>Barra de Progreso</h2>                        
                <progress value="70" max="100" style="width: 300px;">70%</progress>                                  
                <p>Esta barra de progreso indica un progreso del 70%.</p>
                 </body>
                 </html>



Meter

La etiqueta meter se emplea para representar una medida escalar dentro de un rango conocido. A diferencia de progress, que se enfoca en el progreso, meter se usa para mostrar valores de medición como la cantidad de batería restante, la temperatura o el nivel de un recurso.

Sintaxis Ejemplo
<meter> 
                <html>
                 <head>
                <title>Ejemplo de Meter</title>
                </head>
                <body>
                <h2>Medidor</h2>
                                        
                <meter value="0.6" min="0" max="1" low="0.2" high="0.8" optimum="0.5" style="width: 300px;">60%</meter>
                                        
                <p>Este medidor muestra un valor de 60% con una gama de valores bajos, altos y óptimos.</p>
                 </body>
                 </html>



Code

La etiqueta code se emplea para denotar un fragmento de código de computadora dentro de un texto. Esta etiqueta se usa principalmente en documentos técnicos o tutoriales para destacar el código fuente, ya sea en HTML, CSS, JavaScript u otros lenguajes de programación.

Sintaxis Ejemplo
<code> 
            <html>
            <head>
            <title>Ejemplo de Code</title>
            </head>
            <body>
            <h2>Código en Línea</h2>
                                                   
            <p>Aquí hay un ejemplo de código en línea usando la etiqueta <code>.</p>
            <p>El código de ejemplo es: <code>console.log('Hola, mundo!');</code></p>
                                                    
            <h2>Código en Bloque</h2>
            <pre>
            <code>
            function greet() {
            console.log('Hola, mundo!');
             }
            greet();
            </code>
            </pre>
            </body>
            </html>



Kbd

La etiqueta kbd se utiliza para representar la entrada del teclado del usuario en un documento HTML. Es útil para mostrar qué combinaciones de teclas deben presionarse para realizar una acción o comando específico. El texto dentro de kbd se presenta típicamente en una fuente monoespaciada para emular el aspecto de las teclas del teclado

Sintaxis Ejemplo
<kbd> 
            <html>
            <head>
            <title>Ejemplo de KBD</title>
            </head>
            <body>
            <h2>Entrada del Teclado</h2>
                                                    
            <p>Para copiar y pegar texto, presiona <kbd>Ctrl</kbd> + <kbd>C</kbd> para copiar y 
            <kbd>Ctrl</kbd> + <kbd>V</kbd> para pegar.</p>
                                                    
            <p>Para guardar un archivo, utiliza <kbd>Ctrl</kbd> + <kbd>S</kbd>.</p>
            </body>
            </html>



Samp

La etiqueta samp se usa para representar una muestra de salida de un programa o una aplicación. Es comúnmente utilizada en documentación técnica para mostrar el resultado o la salida de un código de ejemplo. El contenido dentro de samp generalmente se muestra en una fuente monoespaciada, similar a la usada para el código, para mantener la claridad en la presentación de los datos de salida.

Sintaxis Ejemplo
<samp> 
                <html>
                <head>
                <title>Ejemplo de Samp</title>
                </head>
                <body>
                <h2>Salida de Ejemplo</h2>
                                                    
                <p>Aquí se muestra un ejemplo de salida de un programa usando la etiqueta <samp>.</p>
                <p>La salida del programa es: <samp>Error: archivo no encontrado</samp>.</p>
                                                    
                <p>Otra salida del programa es: <samp>Conexión exitosa</samp>.</p>
                </body>
                </html>



Var

La etiqueta var se usa para denotar una variable en un contexto matemático o de programación dentro de un documento HTML. Permite resaltar el texto que representa una variable en fórmulas o fragmentos de código, ayudando a la comprensión del contenido técnico. El texto dentro de var se muestra en cursiva por defecto, lo que ayuda a distinguirlo visualmente del resto del texto

Sintaxis Ejemplo
<var> 
            <html>
            <head>
            <title>Ejemplo de Var</title>
            </head>
            <body>
            <h2>Variable</h2>
                                                    
            <p>En el siguiente ejemplo, <var>x</var> es una variable en un cálculo matemático.</p>
            <p>La ecuación es: <var>x</var> + 10 = 20, por lo tanto, <var>x</var> = 10.</p>
                                                    
            <p>Otra variable en el contexto es <var>y</var>, que se define como <var>y</var> = <var>x</var> + 5.</p>
            </body>
            </html>
