Etiquetas para dividir secciones





Div

Se utiliza para agrupar bloques de contenido y aplicar estilos o scripts de forma conjunta. Es un contenedor genérico que no tiene un significado semántico específico, pero es muy útil para estructurar y organizar el contenido en una página web. Los desarrolladores utilizan (div) para crear secciones en el diseño, lo que facilita la manipulación del diseño mediante CSS.

Sintaxis Ejemplo
<Div> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Div</title>
                                    </head>
                                    <body>
                                        <div>
                                            <p>Este es un párrafo dentro de un div.</p>
                                        </div>
                                    </body>
                                    </html>



Section

Se utiliza para definir secciones temáticas dentro de un documento HTML. Cada section representa un área relacionada del contenido, que puede contener un encabezado (h1, h2, etc.) y otros elementos. Esta etiqueta tiene un significado semántico, lo que significa que ayuda a los motores de búsqueda y a los lectores de pantalla a entender la estructura del contenido y su propósito.

Sintaxis Ejemplo
<Section> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Section</title>
                                    </head>
                                    <body>
                                        <section>
                                            <h1>Sección 1</h1>
                                            <p>Contenido de la primera sección.</p>
                                        </section>
                                    </body>
                                    </html>



Article

La etiqueta aside se utiliza para definir contenido que está relacionado de manera secundaria con el contenido principal de la página. Generalmente, se utiliza para incluir información complementaria, como barras laterales, notas, enlaces relacionados, o información adicional que no es parte del flujo principal del contenido. Esta etiqueta tiene un significado semántico, ayudando a los motores de búsqueda y a los lectores de pantalla a entender la relación del contenido.

Sintaxis Ejemplo
<Article> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Article</title>
                                    </head>
                                    <body>
                                        <article>
                                            <h1>Artículo Principal</h1>
                                            <p>Este es el contenido de un artículo.</p>
                                        </article>
                                    </body>
                                    </html>



Header

La etiqueta header se utiliza para definir un encabezado para un documento o una sección dentro de un documento HTML. Generalmente, contiene elementos de navegación, logotipos, títulos y otros elementos que presentan o introducen el contenido. Esta etiqueta tiene un significado semántico y ayuda a los motores de búsqueda y a los lectores de pantalla a comprender mejor la estructura y la jerarquía del contenido.

Sintaxis Ejemplo
<Header> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Header</title>
                                    </head>
                                    <body>
                                        <header>
                                            <h1>Encabezado de la Página</h1>
                                        </header>
                                    </body>
                                    </html>



Footer

La etiqueta footer se utiliza para definir un pie de página para un documento o una sección dentro de un documento HTML. Generalmente, contiene información relacionada con el autor, derechos de autor, enlaces a políticas de privacidad, información de contacto y cualquier otra información relevante que se considere apropiada para el final de la página o sección. Esta etiqueta tiene un significado semántico, ayudando a los motores de búsqueda y a los lectores de pantalla a entender mejor la estructura del contenido.

Sintaxis Ejemplo
<Footer> 
                                    <html>
                                    <head>
                                        <title>Ejemplo de Footer</title>
                                    </head>
                                    <body>
                                        <footer>
                                            <p>Este es el pie de página.</p>
                                        </footer>
                                    </body>
                                    </html>



Nav

La etiqueta nav se utiliza para definir un conjunto de enlaces de navegación dentro de un documento HTML. Su propósito principal es agrupar enlaces que permiten a los usuarios navegar por el contenido de la página o entre diferentes páginas del sitio web. Esta etiqueta tiene un significado semántico que ayuda a los motores de búsqueda y a los lectores de pantalla a identificar la sección de navegación de un documento.

Sintaxis Ejemplo
<Nav> 
                                            <html>
                                            <head>
                                                <title>Ejemplo de Nav</title>
                                            </head>
                                            <body>
                                                <nav>
                                                    <a href="index.html">Inicio</a> |
                                                    <a href="about.html">Acerca de</a> |
                                                    <a href="contact.html">Contacto</a>
                                                </nav>
                                            </body>
                                            </html>



Main

La etiqueta main se utiliza para definir el contenido principal de un documento HTML. Este contenido es el que está directamente relacionado con el tema central de la página y se considera el contenido más importante. La etiqueta main ayuda a mejorar la accesibilidad y la semántica del documento, facilitando a los motores de búsqueda y a los lectores de pantalla identificar la sección principal del contenido.

Sintaxis Ejemplo
<
Main>
 
                                                <html>
                                                <head>
                                                    <title>Ejemplo de Main</title>
                                                </head>
                                                <body>
                                                    <main>
                                                        <h1>Contenido Principal</h1>
                                                        <p>Este es el contenido principal de la página.</p>
                                                    </main>
                                                </body>
                                                </html>
